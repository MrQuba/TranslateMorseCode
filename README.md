# TranslateMorseCode
Program for CTF 309 (https://ctflearn.com/challenge/309)
But can be used for translating any sentence from morse code to english alphabet (remember to place '/' at the end of each letter). Example (flag from ctf): ..-./.-../.-/--./.../.-/--/..-/./.-../--/---/.-./..././../.../-.-./---/---/.-../-.../-.--/-/...././.--/.-/-.--/../.-../../-.-/./-.-./..../././.../
